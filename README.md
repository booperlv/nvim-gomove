# nvim-gomove

A complete plugin for moving and duplicating blocks and lines, with complete fold handling, reindenting, and undo-ing in one go.

## Features

### Moving and Duplicating (Vertically and Horizontally):
- Lines in Normal Mode
- Lines in Visual Mode
- Blocks in Normal Mode
- Blocks in Visual Mode

### With the following additional features:
- Full Fold Handling
- Undoing in one go
- Deleting Trailing Whitespaces (Block Vertical)
- Reindenting
